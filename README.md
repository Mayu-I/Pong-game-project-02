## How to play

	 'r' key restarts game

	 'space' key pauses game
  
Left Player :
      
	- 'w' key moves the paddle upside

	- 's' key moves the paddle downside
  
	- 'd' key shots fire
      
Right player :

	- 'UP Arrow' key moves the paddle upside

	- 'DOWN Arrow' key moves the paddle downside
  
	- 'LEFT Arrow' key shots fire
